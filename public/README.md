# public/

Drop static assets here. Served at the site root.

Add before shipping to production:

- **`favicon.ico`** — AI Intime favicon
- **`og-preview.png`** — 1200×630 Open Graph preview used by the README and social link cards
- **`logo.svg`** — AI Intime logo (white, for dark backgrounds)

None of these are required for `npm run dev` to work.
